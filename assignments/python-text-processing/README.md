# 📘 Assignment: Python Text Processing

## 🎯 Objective

文字列操作、ファイル入出力、基本的なテキスト加工を通じて、実用的なテキスト処理スクリプトを作成できるようになる。

## 📝 Tasks

### 🛠️	ログのクリーンアップと集計

#### Description
生徒は、与えられたテキストログ（複数行文字列）を読み込み、不要な空白や記号を整形したうえで、単語数や行数を集計する。

#### Requirements
Completed program should:

- 先頭・末尾の空白を除去して正規化した行を出力できる
- 総行数と総単語数を計算して表示できる
- 空行は集計対象から除外できる


### 🛠️	ファイルからの単語頻度カウント

#### Description
生徒は、テキストファイルを読み込み、単語の出現回数を数えて上位の単語を表示する。

#### Requirements
Completed program should:

- 大文字・小文字の違いを無視して集計できる
- 句読点などの記号を取り除いて単語を分割できる
- 出現回数が多い順に上位5件を表示できる
